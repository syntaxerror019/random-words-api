Random Word API

A simple API that returns random words and there definitions. There are multiple ways to customize the responses it returns. Take a look below:



Example Usage:

/random - Get a random word of any length and the definition.

/random?length=5 - Get a random word with 5 letters and the definition

/random?count=5 - Get 5 random words of any length and their definitions.

/random?length=5&count=5 - Get 5 random words with 5 letters in each the and their definitions.
